# My dotfiles

## Installation
```
cd ~
git clone --bare https://github.com/jchoffmann/dotfiles.git .dot
source .zshrc
dot checkout
dot config --local status.showUntrackedFiles no
```

See also: https://developer.atlassian.com/blog/2016/02/best-way-to-store-dotfiles-git-bare-repo/
